# Vuetiful Wordbook

[![React](https://img.shields.io/badge/vue--cli-3.7.0-brightgreen.svg?logo=Vue)](https://vuejs.org/)

Personal image convertable simple web wordbook powered by Vue.js

## Install

```
npm install
cd ./backend
npm install
cd ../frontend
npm install
```

## Usage

```
npm start
```
